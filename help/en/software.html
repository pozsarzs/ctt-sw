II. Software

With the program, the characteristics of bipolar transistors can be measured, their input and output characteristics can be drawn and selected based on h21e.

Operating modes

Operation mode M0Stand-by
Operation mode M1Measuring collector-emitter breakdown voltage
Operation mode M2Measuring collector-base breakdown voltage
Operation mode M3Measuring emitter-base cutoff current
Operation mode M4Measuring collector-base cutoff current
Operation mode M5Measuring collector-emitter cutoff current
Operation mode M6Measuring and plotting Ube/Ib characteristic curve
Operation mode M7Measuring and plotting Uce/Ic characteristic curves
Operation mode M8Selection by h21e

The software consists of two parts: the user program with a graphical interface and a command-line program that communicates with the hardware. This program only runs during the measurement. The data is transferred from the management program via command line parameters, from the background program with a temporary file (/tmp/ctt.tmp).
